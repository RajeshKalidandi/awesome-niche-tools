## 🚀 Top Tools Curated

### 1. [BB-Browser](https://github.com/epiral/bb-browser) — ⭐ 5,570
**Your browser is the API.** CLI + MCP server for AI agents to control Chrome with your real login state. Solves the "headless browser = no auth cookies" problem.

> Why top: At 5.6K stars with ~50 stars/day velocity, BB-Browser is the most rapidly adopted "agent-uses-your-real-browser" pattern. With MCP support it's directly drop-in for Claude Code, Codex, and any Hermes-style agent.

### 2. [Headroom](https://github.com/chopratejas/headroom) — ⭐ 2,063
**Compress tool outputs 60-95% before they reach the LLM.** Python lib, HTTP proxy, and MCP server. Created May 2026, already at 2K stars.

> Why top: Token economy is the #1 bottleneck for agent deployments. Headroom's MCP server integration means it works with any agent framework out of the box — 91/100 relevance score.

### 3. [Odysseus](https://github.com/odysseus-ai/odysseus) — ⭐ 7,877
**Self-hosted AI workspace** with chat, agents, deep research, email triage, calendar. The most complete local-first ChatGPT alternative to date. (85/100 relevance, HIGH confidence)

### 4. [CodexBar](https://github.com/steipete/CodexBar) — ⭐ 13,911
**Show usage stats for OpenAI Codex and Claude Code without logging in.** Swift macOS menu bar app. By Peter Steinberger (steipete). Massive adoption at 13.9K stars.

### 5. [summarize](https://github.com/steipete/summarize) — ⭐ 6,075
**Point at any URL/YouTube/Podcast or file. Get the gist.** CLI and Chrome extension by steipete. 6K stars, MIT licensed.

## 🧩 Patterns Found

### Pattern: MCP-First Integration
32% of curated tools now ship MCP servers as first-class interfaces. Tools that don't ship MCP are becoming invisible to the agent ecosystem. **Implication:** When building a new dev tool, ship an MCP server on day one — not as an afterthought.

### Pattern: Token Economy Layer
Multiple tools (Headroom, tokentally, [others]) address the same problem: optimize every token before it reaches the LLM. The bottleneck has shifted from "can the model do this?" to "can we afford to ask?"

### Pattern: Local-First AI Workspace
Odysseus, HTML Anything, VTCode, breathe-cli — all work entirely offline with local models. Signals demand for private, customizable AI stacks that don't phone home.
